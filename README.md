# Proyecto-3-Gestion

How to Play:
1. Confirmar dificultad del juego en Menú de obstáculo y Menú de viento 
apretanto numeros según lo que digan las opciones en el menú.
2. Hacer click en el iniciar juego
3. Flechas hacia arriba y abajo para ajustar el ángulo de disparo.
4. Flechas hacia los lados para ajustar la velocidad del disparo
5. ENTER para disparar
